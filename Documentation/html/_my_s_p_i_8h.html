void MySPI_Init ( SPI_TypeDef *  SPI)

Configure le SPI spécifié : FSCK = 281kHz, Repos SCK = '1', Front actif = up Gestion /CS logicielle à part, configure les 4 IO.

  • SCK, MOSI : Out Alt push pull
  • MISO : floating input
  • /NSS (/CS) : Out push pull
    Parameters
    SPI_TypeDef* SPI : SPI1 ou SPI2
